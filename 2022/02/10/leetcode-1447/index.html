LeetCode每日一题-1447. 最简分数（2022-2-10）

2022-02-10 陈科衡 538字 2分钟

题目

题目

给你一个整数 n ，请你返回所有 0 到 1 之间（不包括 0 和 1）满足分母小于等于 n 的 最简 分数 。分数可以以 任意 顺序返回。

其中： 1 <= n <= 100

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/simplified-fractions
著作权归领扣网络所有。

示例 1：

输入：n = 2
输出：[“1/2”]
解释：”1/2” 是唯一一个分母小于等于 2 的最简分数。

示例 2：

输入：n = 4
输出：[“1/2”,”1/3”,”1/4”,”2/3”,”3/4”]
解释：”2/4” 不是最简分数，因为它可以化简为 “1/2” 。

题解

这一题并不难，关键在于 最简分数 ，在于如何排除分子分母可以相互约去的分数；

数学法

利用最大公约数的办法，求分子分母的最大公约数；如果最大公约数是1，则证明分子分母不可以相互约去，是最简分数。
求最大公约数（Greatest Common Divisor，缩写为GCD）可以使用gcd函数，有的语言内置 gcd 函数 ；
js里没有，需要创建一个gcd函数来求最大公约数（辗转相除法）；
